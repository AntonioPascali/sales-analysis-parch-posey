# Sales Analysis Project (SQL + Python)

## Objective
This project analyzes customer sales data to identify top clients, revenue trends, and key business KPIs.

## Tools Used
- SQL
- Python (pandas, matplotlib)
- Excel

## Dataset
Parch & Posey sample database

## What I did
- Extracted data using SQL JOIN queries
- Calculated total revenue per customer
- Analyzed monthly sales trends
- Created visualizations with Python
- Exported results to Excel

## Key Insights
- Identified top revenue-generating customers
- Observed monthly sales evolution
- Highlighted customer value distribution

## Output
- Sales trend chart
- Top customers Excel file
